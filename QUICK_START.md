# QUICK START Guide for OpenLocal AI

## Introduction
OpenLocal AI is an AI platform designed for local deployments for enhanced privacy and performance.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/shatarudra92/ofline_api_02.git
   cd ofline_api_02
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

## Basic Commands
- Start the server:
  ```sh
  python run_server.py
  ```
- Test the model:
  ```sh
  python test_model.py
  ```

## Troubleshooting
- **Issue:** Server fails to start.
  - **Solution:** Ensure the dependencies are correctly installed and that no other services are using the same port.

- **Issue:** Model not responding.
  - **Solution:** Check if the model is properly loaded and the input format is correct.
